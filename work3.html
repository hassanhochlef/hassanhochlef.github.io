Info

This is a desktop application developped with C and with the help of Glade tool, this application was made to manage a farm modernly with different sections : Equipments , Workers, animals and sensors.

The Challenge

As we all know farmers still manage their farms by phone calls and papers, by time things gets heavier and heavier and lot of informations gets lost.

The Solution

We tried to solve farmer's problems by making this desktop application where the user has the ability to mark and keep track of his goods in a simple application with different sections. This application is easy to use and manage also it saves time and money.
